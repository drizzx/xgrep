# xgrep v0.3 Implementation Plan — CSV/TSV support + context lines

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Ship two orthogonal features — (1) auto-walk `.csv`/`.tsv` files with workbook-style A1 output, encoding override via `-E`, RFC 4180 quoting; (2) rg-aligned `-A`/`-B`/`-C` context lines emitted as new `Context`/`Separator` `MatchEvent` variants with a `[context]` tag — without regressing the v0.2.2 perf gates.

**Architecture:** Two independent slices. CSV slice = new `src/reader/csv.rs` module + extension dispatch inside `read_cells` + walker extension expansion + new `-E ENCODING` CLI flag. Context slice = new `ContextOptions` struct + `MatchEvent::Context`/`MatchEvent::Separator` variants + three-pass collect-expand-emit pipeline inside `search_file` (gated on `ctx.is_zero()` so default-flag callers stay on the single-pass path). Printer changes are shared: suppress `Sheet!` prefix when `sheet` is empty (CSV), render `[context]` tag and `--` separator (context). README updated to drop the obsolete "trait-ready" claim.

**Tech Stack:** Rust 2021, two new deps: `csv = "1.3"` (RFC 4180 parser) and `encoding_rs = "0.8"` (WHATWG encoding labels including UTF-16 BOM auto-detect, GBK, Big5, Windows-1252, etc.). Existing: clap 4 (CLI), serde (JSON), ignore + globset (walker), calamine (xlsx — unchanged).

```rust
//! CSV / TSV reader. Returns `CellRecord`s with the same shape the xlsx reader
//! produces, so `search_file` and the printer treat both formats uniformly.
//!
//! Delimiter is chosen by extension (`.csv` = comma, `.tsv` = tab). Encoding is
//! UTF-8 by default with BOM auto-detect for UTF-8, UTF-16LE, UTF-16BE; an
//! `opts.encoding` override (from `-E ENCODING`) supersedes the BOM hint.
//! RFC 4180 quoting via the `csv` crate's `flexible(true)` mode.

use std::fs;
use std::path::Path;

use crate::cell::to_a1;
use crate::config::Layer;
use crate::error::SearchError;
use crate::reader::{CellRecord, ReaderOptions};

pub fn read_csv_cells(
    path: &Path,
    opts: &ReaderOptions,
) -> Result<Vec<CellRecord>, SearchError> {
    let bytes = fs::read(path).map_err(SearchError::Io)?;
    let text = decode(&bytes, opts.encoding.as_deref())?;

    let delimiter = match path.extension().and_then(|s| s.to_str()) {
        Some(ext) if ext.eq_ignore_ascii_case("csv") => b',',
        Some(ext) if ext.eq_ignore_ascii_case("tsv") => b'\t',
        Some(other) => {
            return Err(SearchError::Parse(format!(
                "internal: unexpected extension '{other}' routed to csv reader"
            )))
        }
        None => return Err(SearchError::Parse("file has no extension".into())),
    };

    let mut rdr = ::csv::ReaderBuilder::new()
        .has_headers(false)
        .flexible(true)
        .delimiter(delimiter)
        .quoting(true)
        .from_reader(text.as_bytes());

    let mut out = Vec::new();
    for (row_idx, rec) in rdr.records().enumerate() {
        let rec = rec.map_err(|e| {
            SearchError::Parse(format!("csv row {}: {e}", row_idx + 1))
        })?;
        for (col_idx, field) in rec.iter().enumerate() {
            if field.is_empty() {
                continue;
            }
            out.push(CellRecord {
                sheet: String::new(),
                cell: to_a1(row_idx as u32, col_idx as u32),
                layer: Layer::Display,
                text: field.to_string(),
            });
        }
    }
    Ok(out)
}

/// Decode bytes per the encoding pipeline:
/// 1. BOM detection → hint (utf-8 / utf-16le / utf-16be).
/// 2. Precedence: explicit `override_enc` > BOM hint > default "utf-8".
/// 3. `encoding_rs::Encoding::for_label(label)`; unknown label → Parse error.
/// 4. Decode with malformed-byte replacement (U+FFFD) — same as rg.
fn decode(bytes: &[u8], override_enc: Option<&str>) -> Result<String, SearchError> {
    let (bom_hint, body_start): (Option<&str>, usize) = match bytes {
        [0xEF, 0xBB, 0xBF, ..] => (Some("utf-8"), 3),
        [0xFF, 0xFE, ..] => (Some("utf-16le"), 0),
        [0xFE, 0xFF, ..] => (Some("utf-16be"), 0),
        _ => (None, 0),
    };

    let enc_label = override_enc.unwrap_or_else(|| bom_hint.unwrap_or("utf-8"));

    let encoding = encoding_rs::Encoding::for_label(enc_label.as_bytes())
        .ok_or_else(|| SearchError::Parse(format!("unknown encoding: {enc_label}")))?;

    let (cow, _used, _had_errors) = encoding.decode(&bytes[body_start..]);
    Ok(cow.into_owned())
}
```

```rust
/// Three-pass context emit: given the full `cells` vector (in reader order)
/// and a flat list of (sheet, row) match positions, returns Match / Context /
/// Separator events with `--` inserted between disjoint spans within a sheet
/// AND at sheet boundaries.
///
/// Returned events use `path: PathBuf::new()`, `submatches: Vec::new()`, and
/// `layer: c.layer.as_str().to_string()`. The caller (`search_file`) re-fills
/// `path` and `submatches`.
pub(crate) fn emit_with_context(
    cells: &[crate::reader::CellRecord],
    match_rows: &[(String, u32)],
    ctx: ContextOptions,
) -> Vec<MatchEvent> {
    use std::collections::{BTreeSet, HashMap, HashSet};

    // Pass 2: union (match_row ± N) per sheet.
    let mut context_rows: HashMap<String, BTreeSet<u32>> = HashMap::new();
    for (sheet, r) in match_rows {
        let lo = r.saturating_sub(ctx.before).max(1);
        let hi = r.saturating_add(ctx.after);
        let set = context_rows.entry(sheet.clone()).or_default();
        for rr in lo..=hi {
            set.insert(rr);
        }
    }
    // Match set (for quick is_match lookup in pass 3).
    let match_set: HashSet<(String, u32)> = match_rows.iter().cloned().collect();

    // Pass 3: walk cells in order, emit events. Track `prev` per sheet to
    // insert Separator when (a) the row gap > 1 within the same sheet, or
    // (b) we transition between sheets.
    let mut out: Vec<MatchEvent> = Vec::new();
    let mut prev: Option<(String, u32)> = None;
    for c in cells {
        let r = match crate::cell::row_from_a1(&c.cell) {
            Some(r) => r,
            None => continue,
        };
        let in_ctx = context_rows
            .get(&c.sheet)
            .map_or(false, |s| s.contains(&r));
        if !in_ctx {
            continue;
        }

        // Decide if we need a separator before this cell.
        if let Some((ref prev_sheet, prev_row)) = prev {
            let need_sep = prev_sheet != &c.sheet || r > prev_row + 1;
            if need_sep {
                out.push(MatchEvent::Separator);
            }
        }
        prev = Some((c.sheet.clone(), r));

        let layer = c.layer.as_str().to_string();
        if match_set.contains(&(c.sheet.clone(), r)) {
            out.push(MatchEvent::Match {
                path: std::path::PathBuf::new(),
                sheet: c.sheet.clone(),
                cell: c.cell.clone(),
                layer,
                text: c.text.clone(),
                submatches: Vec::new(),
            });
        } else {
            out.push(MatchEvent::Context {
                path: std::path::PathBuf::new(),
```
